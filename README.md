# Notas:
Configuración en Node:

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     La base de la tabla de multiplicar          [number] [required]
  -h, --hasta    El límite de la multiplicación           [number] [default: 10]
  -l, --listar   Imprime la tabla en consola          [boolean] [default: false]
```
